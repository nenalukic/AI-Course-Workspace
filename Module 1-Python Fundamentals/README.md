# Module 1: Python Fundamentals

## Overview
Module 1 contains the foundational Python exercises. 

All code is configured to run locally in Visual Studio Code using a standard Python virtual environment (`venv`).

## File Structure
* **`lesson1.py`** - **Variables and Data Types:** Demonstrates dynamic typing with Strings, Integers, Floats, and Booleans.
* **`lesson2.py`** - **Control Flow and Loops:** Covers decision-making (`if/elif/else`) and iteration (`for` loops) to process multiple items.
* **`lesson3.py`** - **Data Structures:** Explores how to store collections of data using zero-indexed Lists and key-value Dictionaries.
* **`lesson4.py`** - **Functions and Exceptions:** Bundles logic into reusable functions and implements `try...except` blocks for safe error handling.
* **`project1.py`** - **Amazon Review Analyzer:** The module's capstone mini-project.

## Setup and Execution
To run the scripts in this module, ensure your virtual environment is activated in your VS Code terminal.

**1. Activate the Environment**
Depending on your operating system, run one of the following commands in the terminal:
* **Windows:** `.\venv\Scripts\activate`
* **Mac/Linux:** `source venv/bin/activate`

*(You should see `(venv)` appear at the start of your terminal prompt).*

**2. Run a Script**
Once the environment is active, you can execute any Python file by passing it to the `python` command:
```bash
python project1.py
