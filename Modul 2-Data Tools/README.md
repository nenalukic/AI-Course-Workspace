# Module 2: Data Tools in VS Code

## Overview
This module transitions from standard Python scripts to Jupyter Notebooks for Exploratory Data Analysis (EDA). It covers the foundational data science stack: **NumPy**, **Pandas**, **Matplotlib**, and **Seaborn**, running entirely within Visual Studio Code.

## Setup Instructions
Before running the notebooks, ensure your virtual environment (`venv`) is activated, then install the required packages:
```bash
pip install ipykernel numpy pandas matplotlib seaborn
