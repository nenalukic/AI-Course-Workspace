### Module 4: Programming Transformers

Welcome to the bleeding edge of AI. Transformers are the architecture powering modern Large Language Models (LLMs) like ChatGPT, Claude, and Gemini. In this module, we will learn how they process human language and how to implement them.

#### Prerequisite: Installing the Transformers Library
We will use Hugging Face's `transformers` library, which is the gold standard for working with these models.
Ensure your `venv` is active and run:
```bash
pip install transformers
